# Coding Standard

## JavaScript

- ES6+
- const / let
- Tidak menggunakan var
- Gunakan Template Literal
- Gunakan Arrow Function bila sesuai

## CSS

- Tidak inline CSS
- Gunakan CSS Variable
- Mobile First

## HTML

- Semantic HTML
- Accessibility
- Responsive

## Apps Script

- Semua akses google.script.run melalui GAS Service
