## Tentang

TKR Engine V2 adalah framework yang dirancang untuk membangun bahan ajar interaktif SMK secara cepat, konsisten, dan modern.

Framework ini mendukung:

- Modul Digital
- Quiz Interaktif
- Dashboard Guru
- Teaching Factory
- Learning Progress
- Google Spreadsheet
- Dark Mode
- Bookmark
- Search
- Responsive Design
